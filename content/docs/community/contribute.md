# Contribute

StoneYield welcomes contributions from the community. Help us build the future of decentralized yield protocols.

## GitHub Repository

**[github.com/stoneyield](https://github.com/stoneyield)**

Visit our GitHub to:
- View source code
- Report bugs and issues
- Submit pull requests
- Contribute to development

## Ways to Contribute

### Code Contributions
- Smart contract improvements
- Frontend enhancements
- Bug fixes
- Gas optimizations

### Documentation
- Fix typos and errors
- Add examples and tutorials
- Improve clarity
- Translate content

### Testing
- Report bugs
- Test new features
- Provide UX feedback

### Community
- Answer questions on Discord/Telegram
- Create educational content
- Share protocol updates
